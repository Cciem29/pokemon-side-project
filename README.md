# Pokémon Side Project

This project explores and analyzes Pokémon stats to see any trends or if predictions can be made.

## Features

- Clustering Pokémon into classes (Tank, Underpowered, Fast Sweeper, and balanced)
- K-Nearest Neighbors classification
- XGBoost classification with hyperparameter tuning
- Damage calculation simulator (Rough Draft, very basic)

## Data

- Stats used: HP, Attack, Defense, Sp. Atk, Sp. Def, Speed, BST (Base Stat Total)
- Pokémon type (Type 1)

## How to Run

1. Install required Python packages
2. Run Jupyter notebooks or scripts
3. Explore results and visualizations
